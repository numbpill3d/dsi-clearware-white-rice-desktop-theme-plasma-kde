# DSi (clearware) White Rice — Desktop Theme for KDE Plasma

A full desktop theme suite inspired by the **Nintendo DSi** menu UI.
White/grey plastic aesthetic, rounded corners, glossy Y2K sheen, DSi-blue accents.

---

## Contents

| Folder | KDE Store Category |
|---|---|
| `dsi-window-decoration/` | Window Decorations (Aurorae) |
| `dsi-plasma-theme/` | Plasma Styles |
| `dsi-color-scheme/` | Color Schemes |
| `dsi-icon-theme/` | Icon Themes |
| `dsi-cursor-theme/` | Cursors |

---

## Installation

### Window Decoration
Copy `dsi-window-decoration/` to `~/.local/share/aurorae/themes/dsi/`
then select **DSi** in System Settings → Appearance → Window Decorations.

### Plasma Style
Copy `dsi-plasma-theme/` to `~/.local/share/plasma/desktoptheme/dsi/`
then select **DSi** in System Settings → Appearance → Plasma Style.

### Color Scheme
Copy `dsi-color-scheme/DSi.colors` to `~/.local/share/color-schemes/`
then select **DSi** in System Settings → Appearance → Colors.

### Icon Theme
Copy `dsi-icon-theme/` to `~/.local/share/icons/dsi/`
then select **DSi** in System Settings → Appearance → Icons.

### Cursor Theme
Copy `dsi-cursor-theme/` to `~/.local/share/icons/dsi-cursors/`
then select **DSi Cursors** in System Settings → Appearance → Cursors.
